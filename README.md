# 钓鱼摸鱼游戏项目文档

## 项目概述

一款纯文字挂机钓鱼游戏，适合工作摸鱼场景，支持网页版运行。

## 文件结构

```
fishing/
├── index.html      # 主页面结构
├── style.css       # 样式设计
├── game.js         # 核心游戏逻辑
└── gameData.json   # 游戏配置数据
```

## 核心功能

### 1. 钓鱼系统
- 自动挂机钓鱼，上钩时间固定5秒（无鱼饵时翻倍）
- 小游戏判定：5个WASD按键顺序输入
- 5秒内完成：闪耀概率+2%，不消耗鱼饵
- 30秒内完成：不消耗鱼饵
- 失败/超时：消耗鱼饵

### 2. 水域系统（3个）
| 水域 | 主要鱼种 | 唯一掉落 |
|------|---------|---------|
| 静默废湾 | 脉冲鱼50%、辐射鱼30%、纳米鱼15% | 废电池 |
| 热核裂谷 | 裂变鱼45%、辐射鱼25%、脉冲鱼20% | 耐热线圈 |
| 虚空暗礁 | 虚空鳕40%、纳米鱼25%、辐射鱼20% | 虚空粉尘 |

### 3. 鱼类系统
- **6种鱼**：脉冲鱼、裂变鱼、辐射鱼、纳米鱼、虚空鳕、霓虹珍珠鳞（稀有）
- **4种大小**：小型 < 中型 < 巨型 < 大型（价格排序）
- **闪耀属性**：基础2%概率，幸运饵+2%，奇点竿+2%，最大6%，售价×5

### 4. 商店系统
- **鱼竿**：竹竿（初始）、共振钓竿(1000金币)、奇点竿(5000金币，闪耀+2%)
- **鱼饵**：基础饵(4金币)，可合成裂变饵、虚空饵、脉冲饵、幸运饵

### 5. 合成系统
- 基础鱼饵：1任意材料 → 2个
- 裂变饵：废电池×2 → 3个（裂变鱼+40%）
- 虚空饵：耐热线圈×2 → 3个（虚空鳕+40%）
- 脉冲饵：虚空粉尘×2 → 3个（脉冲鱼+40%）
- 幸运饵：废电池+虚空粉尘 → 2个（闪耀+2%）

### 6. 存档管理
- 支持删除、导出、导入存档
- 使用 localStorage 存储，导出为 Base64 编码

### 7. 图鉴系统
- 记录捕获过的鱼类
- 区分普通版和闪耀版

## 游戏循环

```
开始钓鱼 → 上钩触发小游戏 → 成功获得鱼/失败消耗鱼饵 → 
渔获入鱼篓 → 出售/收藏 → 金币购买装备 → 提升效率
```

## 数据配置

所有数值、概率、配方统一存放在 `gameData.json` 中，包括：
- 鱼类属性和价格
- 水域配置和概率
- 鱼竿和鱼饵属性
- 随机事件配置
- 钓鱼等级概率表

## 技术实现

- **前端框架**：原生HTML/CSS/JavaScript
- **数据存储**：localStorage + Base64 导出
- **通知功能**：浏览器 Web Notification API
- **响应式设计**：适配移动端

## 启动方式

```bash
python -m http.server 8080
# 访问 http://localhost:8080
```
